# chess-javaFX
chess programmed using javaFX. <br>
Images for pieces taken from wikipedia. <br>
Currently you have to take the king in order to win. <br>
W.I.P Features <br>
<ul>
  <li>Checkmate</li>
  <li>Stalemate</li>
  <li>Choosing piece to promote into</li>
  <li>Time Control</li>
  <li> Playing with a bot (stockfish) </li>
</ul>
